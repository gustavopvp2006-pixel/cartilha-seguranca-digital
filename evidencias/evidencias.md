# Evidências do Projeto

Esta pasta será utilizada para armazenar evidências relacionadas ao desenvolvimento do projeto.

Entre os registros que podem ser incluídos estão:

* Capturas de tela do repositório no GitHub
* Registros das versões da cartilha
* Resultados dos testes realizados
* Documentos utilizados no desenvolvimento do projeto

Essas evidências permitem demonstrar o progresso do projeto e comprovar a realização das atividades propostas.
